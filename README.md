# Calendar

# What is it
- A simple calendar website that shows the current date and time. You can switch between different months and it highlights the current date.

# Features
- Clock
- Current date
- Month picker
- Icon

# Why I made this
- I made this to practice my HTML, CSS, and JavaScript while creating something useful

# How I made it
- HTML
- CSS
- JavaScript

# Challenges and Learning
- A struggled with generating the days of the months.
- I learned how to create animations and format dates and times in JavaScript.
